# Fivetran CLI 

A simple Fivetran API client for the command-line.

## Dependencies

* [uv](https://docs.astral.sh/uv/getting-started/installation/)
* A Fivetran API Key and Secret

## Installation

### Linux / macOS

```bash
make install
```

## `fivetran` menu

```bash
./fivetran --help
```

## Querying Fivetran API

```bash
export FIVETRAN_APIKEY="YOUR_API_KEY"
